Navigating the canvas
One finger Draw & Select

Long press Context menu
One finger — draw, tap to place, long-press for menu.

Two fingers together — pinch to zoom, spread to pan. Works in any tool.

Hold + tap — hold one finger, tap a second to set a selection box corner. Drag right-to-left for crossing select (green), left-to-right for window select (blue).

Swipe from the left edge to open the menu.
Drawing & precision input
Select a draw tool from the bottom toolbar, then tap to place points.

For precise coordinates, use the popups that appear automatically — tap the number, type with your keyboard, tap confirm.

Select lines by tapping them. Use the crossing drag (right-to-left) to grab everything a box touches.
